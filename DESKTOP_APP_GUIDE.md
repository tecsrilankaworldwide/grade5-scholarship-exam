## 🚀 INSTALLATION

### **Windows:**

1. **Download the installer:**
   - File: `Grade5-Exam-Platform-Setup-2.0.0.exe`
   - Size: ~300 MB

2. **Run the installer:**
   - Double-click the downloaded file
   - Click "Yes" if Windows asks for permission
   - Follow the installation wizard
   - Choose install location (default: `C:\Program Files\Grade5 Exam Platform`)

3. **Launch the application:**
   - Desktop shortcut created automatically
   - Or find in Start Menu: "Grade 5 Exam Platform"
   - Double-click to open

4. **First launch:**
   - Application starts services automatically
   - Wait 30-60 seconds for first-time setup
   - Login page opens automatically

### **Mac:**

1. **Download the installer:**
   - File: `Grade5-Exam-Platform-2.0.0.dmg`
   - Size: ~300 MB

2. **Install:**
   - Open the `.dmg` file
   - Drag the app icon to Applications folder
   - Eject the `.dmg` file

3. **Launch:**
   - Find "Grade 5 Exam Platform" in Applications
   - Double-click to open
   - If Mac says "unidentified developer":
     - Right-click app → "Open"
     - Click "Open" again to confirm

4. **First launch:**
   - Services start automatically
   - Wait 30-60 seconds
   - Application opens to login page

### **Linux:**

1. **Download the installer:**
   - File: `Grade5-Exam-Platform-2.0.0.AppImage` or `.deb`
   - Size: ~300 MB

2. **Install:**

   **For AppImage:**
   ```bash
   chmod +x Grade5-Exam-Platform-2.0.0.AppImage
   ./Grade5-Exam-Platform-2.0.0.AppImage
   ```

   **For .deb (Ubuntu/Debian):**
   ```bash
   sudo dpkg -i Grade5-Exam-Platform-2.0.0.deb
   ```

3. **Launch:**
   - Find in Applications menu
   - Or run from terminal: `grade5-exam-platform`

## 🔧 TROUBLESHOOTING

### **App won't start**
**Windows:**
1. Right-click app icon
2. Select "Run as administrator"
3. Allow in Windows Firewall if prompted

**Mac:**
1. System Preferences → Security & Privacy
2. Allow app to run
3. Try opening again

### **"Services failed to start"**
1. Close the app completely
2. Wait 10 seconds
3. Open again
4. App will attempt automatic recovery

### **Slow first startup**
- First launch takes 1-2 minutes (normal)
- Creating database and loading data
- Subsequent starts are faster (~10 seconds)

### **"Port already in use"**
- Another instance might be running
- Check system tray for app icon
- If found, right-click → Quit
- Start app again

2. Drag "Grade 5 Exam Platform" to Trash
3. Empty Trash

